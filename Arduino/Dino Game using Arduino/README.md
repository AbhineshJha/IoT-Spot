# Dino Game using Arduino

## AIM:
To create a dino game using arduino.

## Purpose:

To mimic the chrome dino game using arduino with LCD and Push buttons.

## Short Description:

**Requirements:**

- Arduino Board
- 16x2 LCD
- 1K ohm Resistor
- 2 Pushbuttons
- Some Wires
