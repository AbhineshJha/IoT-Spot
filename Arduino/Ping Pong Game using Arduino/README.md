# Ping Pong Game using Arduino


## Short Description:

**Requirements:**

- 1 x Arduino UNO board.
- 2 x NeoPixel LEDs.
- 1 x piezoelectric crystal.
- 6 x resistors.
- 5 x pushbuttons.
- 1 x potentiometer.
- 1 x LCD display 16x2.
- 1 x breadboard.
